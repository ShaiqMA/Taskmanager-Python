# Python Task Manager

Een eenvoudige command-line takenbeheerder geschreven in Python.
Met deze applicatie kunnen gebruikers dagelijkse taken beheren via de terminal.

## Over het project

Dit project is ontwikkeld om basisprincipes van Python en data-opslag toe te passen in een praktische applicatie.
Gebruikers kunnen taken toevoegen, bekijken, markeren als voltooid en verwijderen.

Alle taken worden lokaal opgeslagen in een JSON-bestand.

## Functionaliteiten

* Taken toevoegen
* Taken bekijken
* Taken verwijderen
* Taken markeren als voltooid
* Lokale opslag met JSON
* Gebruiksvriendelijke command-line interface


## Gebruikte technologieën

* Python
* JSON
